# NutriSakti: Decentralized Maternal & Child Health Guardian

## 🌿 Overview

**NutriSakti** is an open-source, blockchain-based solution developed for **Eastern Indonesia** (NTT, Papua, Maluku) to combat stunting and ensure social protection for pregnant mothers and infants. We "hack" traditional aid limitations by leveraging local biodiversity and decentralized oracles to provide "verifiable proofs of impact".

## 🚀 Key Features

* 
**BPJS Verification Bridge:** Privacy-preserving integration to identify gaps in national health insurance (BPJS) using "zero-knowledge proofs". Automated alerts to health posts for uncovered mothers.


* 
**Visual Nutrition Hack:** AI-driven identification of local raw materials to generate "on-demand AI videos" for deep-nutrition cooking.


* 
**BGN Quality Auditor:** Nutritional Balance Validator that allows mothers to audit government-provided food quality, creating transparent accountability.


* 
**Impact-Linked Disbursement:** Equity-free funding distribution of "Maternal-Baby Kits" via ETH, BTC, or USDC based on "real-time reconciliation" of health milestones.


* 
**Offline-First Architecture:** Designed for "mobile-friendly scalability," allowing data synchronization for mothers "without internet access".



## 🛠 Technical Stack & Open Source

This project is built on "open-source frontier technology".

* 
**Blockchain:** Polygon PoS (Layer 2) for "lower transaction costs" and fast transactions.


* 
**External Data Verifiers:** Chainlink-powered bridges to bring real-world data on-chain (BPJS insurance, disaster alerts, nutrition standards).


* 
**Emergency Signal Gateway:** Real-time disaster monitoring with automatic priority elevation for pregnant mothers.


* 
**Storage:** IPFS for decentralized storage of AI video assets.
